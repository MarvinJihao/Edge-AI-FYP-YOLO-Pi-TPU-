# Edge AI FYP YOLO Pi TPU

This repository contains the source code used for the final year project on
YOLO-based object detection and edge AI deployment experiments.

## Repository structure

- `yolov5-compare/` - YOLOv5 comparison code and experiment scripts.
- `YOLOv5-Lite-compare/` - YOLOv5-Lite comparison and deployment code.
- `Yolov8-main/` - YOLOv8 training, prediction, and evaluation scripts.
- `compute.py` - top-level metric/utility script.
- `requirements.txt` - Python dependencies used by the project.

## Large files

Large generated artifacts are intentionally excluded from Git:

- datasets
- training runs and logs
- model weights such as `.pt`, `.onnx`, `.tflite`, and `.engine`
- compressed dataset archives
- local IDE/cache/build files

This keeps the GitHub submission lightweight and reproducible. The excluded
files can be regenerated by training/export scripts or provided separately as
external artifacts if required.

## Typical setup

```bash
pip install -r requirements.txt
```

Individual subprojects may also include their own `requirements.txt` files.

## Notes

The committed files are intended to show the implementation, configuration, and
experiment code. Generated outputs and trained model files are omitted to avoid
exceeding GitHub repository size limits.
